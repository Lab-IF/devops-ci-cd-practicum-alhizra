# Harapan Saya dari Praktikum DevOps Ini
Harapan saya dari praktikum DevOps ini adalah dapat memahami konsep DevOps secara nyata, tidak hanya secara teori tetapi juga melalui praktik langsung. Dengan adanya praktikum ini, saya berharap bisa mengetahui bagaimana proses pengembangan aplikasi dapat dilakukan secara lebih terstruktur, mulai dari pembuatan aplikasi, pengujian, hingga proses deployment ke server. Saya juga ingin memahami bagaimana cara kerja tools yang digunakan dalam DevOps, seperti Docker, Docker Compose, dan konsep CI/CD.
Selain itu, saya berharap praktikum ini dapat membantu saya meningkatkan kemampuan teknis, terutama dalam mengelola environment aplikasi agar dapat berjalan dengan konsisten di berbagai kondisi. Praktikum DevOps juga diharapkan dapat melatih saya untuk bekerja lebih disiplin, terstruktur, dan terbiasa menggunakan otomatisasi dalam pengembangan software.
Tidak hanya dari sisi teknis, saya juga berharap praktikum ini dapat meningkatkan pemahaman saya tentang pentingnya kerja sama tim, komunikasi, dan kolaborasi antara developer dan operations. Dengan begitu, ilmu yang saya dapatkan dari praktikum DevOps ini bisa menjadi bekal yang berguna untuk dunia kerja di bidang teknologi di masa depan.

# Skill yang ingin saya kuasai di akhir semester
Di akhir semester, saya ingin menguasai skill dasar DevOps, khususnya penggunaan Docker dan Docker Compose untuk menjalankan aplikasi secara terstruktur. Saya juga ingin memahami alur kerja CI/CD secara sederhana agar proses build dan deployment aplikasi dapat dilakukan dengan lebih efisien. Selain itu, saya berharap bisa lebih terbiasa membaca dokumentasi teknis dan menyelesaikan error yang muncul saat praktikum. Skill ini diharapkan dapat menjadi bekal awal yang bermanfaat untuk dunia kerja di bidang teknologi.
